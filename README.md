### electron部分
> ps. 最新版本的mac无法使用electron交叉编译windows程序。比如macOS Catalina 10.15.1.   

编译程序可通过以下命令： （也可以使用npm run代替yarn）
```
yarn install-build-win
or
yarn install-build-mac
```
构建目录为项目根目录的dist文件夹下demo.exe or demo-0.0.1.dmg。
